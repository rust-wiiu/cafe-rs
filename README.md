# cafe-rs

> [!CAUTION]
> `cafe-rs` is in active experimentation / development phase.

`cafe-rs` is the heart of Rust software for Wii U homebrew. It aims to be the Rust alternative to [wut](https://github.com/devkitPro/wut), _the_ C++ SDK / toolchain for homebrew apps, developed by [devkitPro](https://github.com/devkitPro) and [the community](https://github.com/devkitPro/wut/graphs/contributors).

## Goals

- Easy-to-use: Provide intuitive APIs along with good documentation
- Idiomatic: Write Rust code that feels right
- std-like: Adhere to Rust's [std](https://doc.rust-lang.org/std/) APIs as much as possible where applicable
- Compatible: Create applications within the [existing ecosystem](https://github.com/wiiu-env)
- Simple: Require as few external tools as possible
